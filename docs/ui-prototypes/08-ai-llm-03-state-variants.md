### 3.2 加载状态（≥ 3s 才显示）

**智能卡**：

```
┌─────────────────────────────────────────┐
│ ✦ 今日洞察              [刷新] [ⓘ]      │
├─────────────────────────────────────────┤
│                                         │
│   正在分析今日数据...                   │  ← skeleton / spinner
│   (动画: 3 个跳动的圆点)               │
│                                         │
│  ┌──────┐ ┌──────┐ ┌──────┐            │
│  │▒▒▒▒ │ │▒▒▒▒ │ │▒▒▒▒ │            │  ← 骨架屏
│  │▒▒▒  │ │▒▒▒  │ │▒▒▒  │            │
│  └──────┘ └──────┘ └──────┘            │
│                                         │
└─────────────────────────────────────────┘
```

- 主文区：`el-skeleton :rows="2" animated` 或自定义 "分析中..." + 3 dots 动画
- chip 区：`el-skeleton :rows="3"` 占位
- 刷新按钮：`loading` 旋转 + disabled
- ⓘ 按钮：`disabled`（加载中不能开抽屉）

**延迟阈值**：响应 ≥ 3s 才进入 loading 态；< 3s 用户感知不到。

### 3.3 降级状态（LLM 失败 → 模板）

**用户感知差异**：

| 元素 | LLM 成功 | 模板降级 |
|---|---|---|
| 智能卡 source 标签 | 🟢 "AI 生成" | ⚪ "模板" |
| 智能卡 headline | 2 句（含判断 + 建议）| 1 句（任务 + 消费拼接）|
| 抽屉 source 标签 | 🟢 "AI 生成" | ⚪ "模板" |
| 抽屉内容 | headline + 3 指标 + 来源说明 | 同（headline 略短）|
| 独立分析页 Advice 区 | ✅ 显示 | ❌ **隐藏** |
| 独立分析页 Highlight 区 | ✅ 显示 | ❌ **隐藏** |
| 独立分析页 4 chip | ✅ 显示 | ✅ 显示 |
| 底部"v2.2 将加入..." | ✅ 显示 | ✅ 显示 |

> **关键 UX**：模板降级时用户感知差异最小化（source 标签 + 缺少 advice/highlight）；无错误弹窗、无 loading 延长。
