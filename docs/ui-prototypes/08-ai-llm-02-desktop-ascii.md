```
│                                                                 │
│  生成于 08:30 · 缓存 1h23m 前                                    │
└─────────────────────────────────────────────────────────────────┘
```

**v2.1 改动（vs v2.0）**：

| 元素 | v2.0 | v2.1 |
|---|---|---|
| 标题栏按钮 | `[刷新][查看明细]` | `[AI 生成][刷新][ⓘ]` |
| 顶部标签 | ❌ 无 | ✅ `source` 标签（v2.1 新增）|
| headline | 1 句（任务 + 消费）| 2 句（判断 + 建议，可被 LLM 扩展）|
| 底部时间 | "生成于 08:30" | "生成于 08:30 · 缓存 1h23m 前"（含 freshnessSeconds 反算）|

**布局规则**：

- 桌面 `el-row :gutter="16"` 包裹 3 chip；`el-col :xs=24 :sm=8` 三等分
- 主文 `font-size: 18px; font-weight: 600; line-height: 1.5`
- chip 卡 `padding: 12px 16px; background: var(--el-fill-color-light)`
- source 标签：`<el-tag size="small" :type="source === 'llm' ? 'success' : 'info'" effect="plain">`
  - `source === 'llm'` → 绿色"AI 生成"
  - `source === 'template'` → 灰色"模板"
  - `source == null`（v2.0 旧缓存）→ 不显示标签
- 趋势箭头：UP = `↑` 绿色；DOWN = `↓` 红色；FLAT / NONE = `—` 灰色

**整卡可点击**：点击 headline / chip 区 → 触发抽屉（与 ⓘ 按钮等价）。

**布局规则**：

- 整页容器 `<el-main>` 或 `<div class="ai-analysis">`，`max-width: 960px; margin: 0 auto; padding: 24px`
- 顶部栏：`<el-page-header>` 或简化 `<div>`：返回箭头 + 标题 + 刷新按钮 + source 标签
- Headline 卡片：`<el-card shadow="never">` + `<h1>` 样式
- Advice 卡片：`<el-card shadow="never">` + 💡 icon + `<p>`
- Highlight 卡片：`<el-card shadow="never">` + ✨ icon + `<ul>` bullet 列表
- 关键指标：`<el-row :gutter="16">` + `<el-col :xs=12 :sm=6>` 4 等分
- 底部信息：`<p class="footnote">` 灰字

**source 标签逻辑**（与卡片、抽屉一致）：
- `source === 'llm'` → 绿色"AI 生成"
- `source === 'template'` → 灰色"模板"
- `source == null`（理论上不该出现）→ 不显示

**advice / highlight / mood / llmMeta 渲染规则**：
- LLM 成功 → 全部渲染
- 模板降级 → advice / highlight / mood / llmMeta 均为 `null` → **只渲染 headline 卡片 + 关键指标卡片**（advice/highlight 区块隐藏）
- 空数据 → headline = `headline.empty`，advice/highlight 区块隐藏，关键指标 4 chip 全 `—`
