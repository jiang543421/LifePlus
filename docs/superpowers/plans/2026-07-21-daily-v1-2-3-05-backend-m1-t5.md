- [ ] **Step 1**: 创建 `DietMetricProvider`(**注释硬性约束**)

```java
package com.lifepulse.daily.provider;

import com.lifepulse.daily.DietMetrics;
import org.springframework.stereotype.Component;

import java.time.LocalDate;

/**
 * 饮食模块未上线时的占位 Provider。
 *
 * <p><b>必须保持 enabled=false,直到饮食模块正式接入</b>;前端 DietMetricsCard
 * 永远渲染 disabled 视觉。任何修改此 Provider 以"启用饮食"的尝试,必须先
 * 在 docs/specs/07-diet-design.md 落档变更 + 调整 DietMetricProviderTest 的
 * 锁死断言(否则单测直接失败,PR review 卡住)。
 *
 * <p>饮食模块独立排期,不在 v1.2.3 范围。
 */
@Component
public class DietMetricProvider implements MetricProvider<DietMetrics> {

  /** 锁死开关,绝不接受外部配置注入。 */
  private static final boolean ENABLED = false;
  private static final String REASON = "饮食模块未上线";

  @Override
  public DietMetrics aggregateDaily(long userId, LocalDate date) {
    if (ENABLED) {
      throw new IllegalStateException(
          "DietMetricProvider.enabled 切换到 true 之前必须先解锁 DietMetricProviderTest 的锁死断言");
    }
    return new DietMetrics(false, null, REASON);
  }
}
```

- [ ] **Step 2**: RED — 写**锁死单测**

```java
package com.lifepulse.daily.provider;

import com.lifepulse.daily.DietMetrics;
import org.junit.jupiter.api.*;
import java.time.LocalDate;
import static org.assertj.core.api.Assertions.*;

class DietMetricProviderTest {

  DietMetricProvider provider = new DietMetricProvider();

  @Test void alwaysReturnsDisabled() {
    DietMetrics m = provider.aggregateDaily(1L, LocalDate.of(2026, 7, 15));
    assertThat(m.enabled()).isFalse();
    assertThat(m.value()).isNull();
    assertThat(m.reason()).isEqualTo("饮食模块未上线");
  }

  @Test void reasonIsExplicitNotNull() {
    // 防止有人改成 null reason 让前端不显示
    DietMetrics m = provider.aggregateDaily(999L, LocalDate.now());
    assertThat(m.reason()).isNotBlank();
  }

  @Test void differentUsersAndDatesStillDisabled() {
    // 锁死:任何 userId / 任何 date 都不能触发 enabled=true
    for (long uid : new long[]{1L, 999L, Long.MAX_VALUE}) {
      for (LocalDate d : new LocalDate[]{
          LocalDate.of(2026, 7, 15), LocalDate.of(2025, 1, 1), LocalDate.now()}) {
        assertThat(provider.aggregateDaily(uid, d).enabled())
            .as("uid=%s date=%s", uid, d)
            .isFalse();
      }
    }
  }
}
```
