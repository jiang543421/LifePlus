# LifePulse v2.0 — AI 分析模块

**Release Date**: 2026-07-22  
**Branch**: `feat/ai-v2.0`  
**Tag**: 待打

## 概述

新增首页 AI 分析模块：智能卡片 + 抽屉详情。纯规则 + 模板引擎实现，无 LLM 依赖。

## 用户故事映射

| 故事 | 实现 |
|---|---|
| US-1 查看今日洞察 | AiInsightCard + GET /today |
| US-2 点击查看详情 | AiInsightDrawer |
| US-3 手动刷新 | POST /refresh + 刷新按钮 |
| US-4 数据为空时优雅降级 | "还没有数据" headline + 空 chip |
| US-5 模块降级不影响体验 | daily 默认关 + 单 provider 异常隔离 |

## 端点

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/v1/ai/insight/today` | 当前洞察（缓存优先） |
| POST | `/api/v1/ai/insight/refresh` | 清缓存 + 重算（限流 6/min） |

## 限制

- 单用户每日洞察（非跨用户）
- 仅 3 chip，固定顺序（task/expense/plan）
- 纯规则 + 模板，不支持自然语言生成
- Daily 模块默认关；启用需 `lp.ai.daily-enabled=true`
