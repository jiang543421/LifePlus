## 11. 前端 UX

### 11.1 智能卡（替换 HomeView 占位卡）

```
┌─────────────────────────────────────┐
│ [图标]  AI分析              🔄  ⓘ  │
│                                      │
│ 今日任务完成率 80%，较昨日提升 5     │
│ 个百分点；本周消费 ¥420，较上周下    │
│ 降 12%。                            │
│                                      │
│ ┌──────┐ ┌──────┐ ┌──────┐          │
│ │任务80%│ │¥420  │ │日程4 │          │
│ │+5pp↑│ │-12%↓│ │持平  │          │
│ └──────┘ └──────┘ └──────┘          │
│                                      │
│              查看明细 →              │
└─────────────────────────────────────┘
```

- 整卡可点击（→ 抽屉）
- 🔄 按钮：触发 refresh
- ⓘ 按钮：触发抽屉（与整卡点击等价）
- chip 颜色：up=绿 / down=红 / flat=灰 / none=浅灰

### 11.2 详情抽屉（`AiInsightDrawerView`）

- 标题："AI 分析 · {generatedAt 格式化}"
- 顶部：完整 headline（可能比卡片更长）
- 中部：3 个 chip 大尺寸展示
- 底部："v2.1 将推出更详细的趋势分析" 提示
- 关闭：右上角 X、ESC 键、点击遮罩

### 11.3 空状态

`headline="还没有数据，继续记录几天后将出现洞察。"` + 3 chip 全 `value="—" trend="none"`，无 chip 副标。
