## 1. 模块概述

任务模块是 LifePulse 的「快速记录与勾选」核心，承载三类场景：

1. **随手记录**：1 分钟内创建一条待办，仅填标题也能存。
2. **组织优先级**：按截止日、优先级、标签筛选，决定今天先做哪个。
3. **关联日程**：任务可挂到某条日程事件下（如「客户会议准备」挂到「周三 14:00 客户会议」）。

属于「期望需求」级别的核心模块——是用户每天至少打开一次的高频路径。

## 2. 功能清单（MVP）

### 2.1 任务 CRUD

- [ ] 创建任务（标题必填，可选截止日 / 优先级 / 标签 / 关联到某个日程事件）
- [ ] 编辑任务（标题、截止日、优先级、标签、所属日程）
- [ ] 快速切换任务状态（待办 → 已完成 / 已取消）
- [ ] 删除任务（软删除，可在「已删除」面板中恢复，30 天后清理）

### 2.2 列表与筛选

- [ ] 任务列表（支持按状态 / 优先级 / 标签 / 截止日区间筛选）
- [ ] 任务列表分页（每页 20 条）
- [ ] 任务详情查看
- [ ] 任务列表默认按截止日升序，无截止日的任务排在最末
- [ ] 列表页右上角显示当前筛选条件下「待办」总数

### 2.3 任务 ↔ 日程联动

- [ ] 任务创建表单可选「所属日程」，下拉中仅显示当前用户的日程事件
- [ ] 在某个日程详情页可看到「本日程下的任务」列表，并可一键新建任务并自动关联
- [ ] 删除日程时，关联任务不被删除，仅解除关联关系

### 2.4 首页集成

- [ ] 首页任务卡显示今日待办数（仅「待办」状态、截止日 = 今日）
- [ ] 点击首页任务卡跳转到任务列表页

## 3. 用户故事

### 故事 TASK-1：记录并管理待办
> 作为 **用户**，我需要 **快速记录今天要做的事并打勾完成**，以便于 **获得「完成事项」的反馈感**。

**验收标准（AC）**：
1. 在任务列表页可创建一条任务，仅填标题后保存即出现在列表顶部。
2. 任务列表支持按「全部 / 待办 / 已完成 / 已取消」切换。
3. 点击任务的状态切换按钮后立即更新视图，无需保存按钮。

### 故事 TASK-2：按优先级与截止日组织任务
> 作为 **任务多的用户**，我需要 **按优先级与截止日排序与筛选**，以便于 **先做最重要的、最紧急的**。

**验收标准（AC）**：
1. 任务列表默认按截止日升序，无截止日的任务排在最末。
2. 可按优先级（高 / 中 / 低 / 无）筛选；筛后视图与未筛视图相互独立保留。
3. 列表页右上角显示当前筛选条件下「待办」总数。

### 故事 TASK-3：把任务挂到某个日程事件
> 作为 **日程与任务并重的用户**，我需要 **在创建任务时指定它属于某个日程事件**，以便于 **事后回看某次会议/活动下要做的事**。

**验收标准（AC）**：
1. 任务创建表单可选「所属日程」，下拉中仅显示当前用户的日程事件。
2. 在某个日程详情页，可看到「本日程下的任务」列表，并可一键新建任务并自动关联。
3. 删除日程时，关联任务不被删除，仅解除关联关系。

## 4. KPI

| 指标 | 目标 |
|---|---|
| 任务列表接口 P95 | ≤ 300 毫秒 |
| 任务创建到列表显示端到端 | ≤ 1 秒 |
| 跨用户越权拦截率 | 100% |
| 「今日待办数」首页加载延迟 | ≤ 500 毫秒 |
| 列表默认排序正确率 | 100%（截止日升序 + 无截止日最末） |

## 5. 边界

**In Scope**：任务 CRUD / 状态切换 / 软删除 / 多维度筛选 / 分页 / 详情 / 按所属日程查看 / 首页计数 / 与日程模块的关联。

**Out of Scope**：
- 任务重复（每天/每周循环）—— Phase 2 评估。
- 子任务 / 任务模板 —— 个人工具不需要。
- 任务评论 / 协作 —— 团队功能，超出 MVP。
- 任务附件 —— 数据模型未设计附件表。
